LinguaPal

This NVDA add-on integrates Google Gemini and Groq AI to assist with real-time translations and general chat with AI directly from your NVDA.

Features

  • Translate clipboard content using your selected AI model
  • Chat with AI with context-aware prompts

Hotkeys

  • NVDA+Alt+C: Translate clipboard text
  • NVDA+Alt+G: Open LinguaPal chat window

Settings

Select your AI model, Configure your API key, target translation language, and update options in the add-on settings panel.

Add-on hotkeys can be changed from the Input Gestures dialog.

Checking for updates

LinguaPal checks for updates on NVDA startup if enabled in settings. You can also manually check for updates from the settings dialog. If a newer version is available, the add-on will download and install it automatically. This add-on was built very recently, therefore it's recommended that you frequently update the add-on if there are any updates.
